By the end of this module, you will be able to:

- Understand the benefits of a layered architecture.
- Identify the Clean Architecture role of a given class.
- Implement a Use Case Interactor in an existing implementation of the Clean Architecture.
- Identify where the Dependency Inversion Principle is applied in the Clean Architecture to follow the Dependency Rule.
- Create a set of user stories for your group project.
- Create sketches of the UI elements of your user story for your group project.
- Create use cases for each user story for your group project.
